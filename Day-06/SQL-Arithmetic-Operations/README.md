# Day 06: Mastering SQL Arithmetic Operators (Business Metrics)

### 🎯 Objective
To apply SQL Arithmetic Operators (+, -, *, /, %) on a real-world `orders` table to derive critical business metrics like profit margins, discounts, and final billing amounts without altering the base data.

### 🛠️ Key Concepts Covered:
* **Derived Columns:** Creating temporary calculation columns using the `SELECT` clause.
* **Profit Analysis:** Calculating net gains using the Subtraction (-) operator.
* **Discounting Logic:** Applying percentage-based deductions using Multiplication (*).
* **Financial Health:** Measuring ROI/Profit Margins using Division (/).

### 📊 Practical Scenarios Solved:
1. **Profit Calculation:** `(selling_price - purchase_price)`
2. **Billing Updates:** Adding flat shipping/packing fees using `(+)`.
3. **Marketing Discounts:** Calculating a 10% deduction on selling prices.
4. **Efficiency Metrics:** Calculating the ratio of profit against investment.
5. **Data Filtering:** Using Modulus (%) to identify specific order patterns.

> *"In business, data is just a number. SQL Arithmetic makes that number meaningful."*
